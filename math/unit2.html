When faced with a number like 1024 you can simplify it by turning 1024 into 2 to the power of 10 or 2^10, this can also be solved by using logarithms. When using logarithms, use the number before the ^ and put it behind the log, then put the said big number in the parentheses, then put the number after the ^ on the right side of the equal sign. Ex: log2(1024) =10 Now, we segue into Inequalities, if a number does not have the same as the one on the other side of the inequality, they are not equal, if it is larger, we use >, if the value is smaller, we use <, if we are not sure if the value is greater or equal to the other value, we use =<, if it is less than or equal to said value, we use =>. Ex: 3=3, x=3 to 7, y=3, therefore xy 3>2, 3<4, and if x=9, and y=3 to 9, x=>y
